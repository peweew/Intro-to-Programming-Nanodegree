Understanding of Structured Documents

Computer code is structured. This makes the computer code to be easy orgainized. For instance, in most programming language, we use class and function to define code with certain property.

The web developing language need to be well develop. A website used to have HTML file, CSS file and javascript file. Each file deal with different functions.

The code in HTMl, CSS and javascript file is also organized. For instance, in html file, the content of website is included in the html tags. For example, the paragraph is usually contained in between <p> and </p> tag. In CSS and javacript, it has similar orignization.

DOM and tree-like structure of HTML

The Document Object Model (DOM) is a cross-platform and language-independent convention for representing and interacting with objects in HTML, XHTML, and XML documents. The nodes of every document are organized in a tree structure, called the DOM tree. Objects in the DOM tree may be addressed and manipulated by using methods on the objects.

HTML structure is organized as a tree structure. Each tag has the begin tag and end tag. For instace, <table></table> The tags can be contained in other tag. The root tag is <html></html> This tag contains all the informations of website. Since all the tags are contained in this tree like structure, the HTML is a tree liked language.
